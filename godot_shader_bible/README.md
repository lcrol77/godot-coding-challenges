# Godot Shader Bible

**Goal:** Browse and study shader examples from the Godot Shader Bible assets.

## Controls
- Mouse look in scenes that use the camera movement script
- Movement uses `move_*` or `ui_*` actions when mapped

## Concepts Practiced
- Shader studies across lighting, VFX, and UI materials
- Scene organization for shader showcases
- Camera movement utilities
