# Golf

**Goal:** Prototype drag-and-drop card layout for a golf-style card game.

## Controls
- Left click: select and drag a card
- Right click: cancel drag

## Concepts Practiced
- Drag-and-drop component design
- Card layout and reparenting
- GUI input handling

## Future Ideas
- Implement golf rules and scoring
- Add deck, shuffle, and discard logic
