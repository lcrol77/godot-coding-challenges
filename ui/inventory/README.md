# Inventory

**Goal:** Prototype a drag-and-drop inventory UI with slots and items.

## Controls
- Left click: drag item
- Right click: cancel drag

## Concepts Practiced
- UI slot grids and item data
- Drag-and-drop components
- Z-index and grouping for dragging

## Future Ideas
- Add item stacking and splitting
- Add tooltips and item inspection
- Add swap and snap animations
