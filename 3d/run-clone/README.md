# Run Clone

**Goal:** Build a runner prototype with lane switching and wall rotation.

## Controls
- A/D: move lanes
- Space: jump
- R: reset position

## Concepts Practiced
- CharacterBody3D movement and damping
- Lane switching and wall-based rotation
- Animation tilt and rotation lock

## Future Ideas
- Add obstacles and collectibles
- Add scoring and speed ramping
- Add lane-change VFX
